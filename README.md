# EventPlatform

# 介绍
这是一个开源社区Event智能处理平台，由于采用了消息队列机制，所以可以实现非即时事件的处理和推送。

## 主要应用场景
社区任务指派（issue/PR责任人推荐机器人）
社区标签自动化（issue/PR标签机器人）
社区任务提醒（延时提醒机器人）
还有其他能力
